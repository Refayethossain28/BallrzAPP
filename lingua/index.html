Enable Live AI accuracy

Translations and lessons are powered by Claude so your API key never touches the browser. Two ways to turn it on:

Local proxy — run it once, then open http://localhost:8788:

cd lingua
ANTHROPIC_API_KEY=sk-ant-… node server.mjs

Hosted (cloud) — deploy the linguaAI Firebase function once and the live web link gets AI automatically (see lingua/README.md).

Until either is connected, Lingua runs offline with a built-in starter phrasebook. Audio (🔊) and saved translations work offline too.
