- **⭐ [Imposter — the party game](https://refayethossain28.github.io/BallrzAPP/imposter/)** — the best **pass-and-play** social-deduction game: one phone, **3–12 players**, no account. Two full game modes: **🔤 Word** — everyone shares one secret word except the **imposter**, who bluffs blind; and **📍 Locations** (Spyfall-style) — everyone gets the same place *and* a unique character role to act out, while the **spy** gets neither and has to deduce where they are. Pass the phone so each player privately flips their card (a slick 3D reveal, optional **hold-to-reveal** for shoulder-surf safety), give a **one-word clue** in the dealt turn order (optionally **two clue rounds**), run the **discussion timer**, then the table agrees on one accusation and votes. Unmask the imposter/spy and the crew scores — unless the caught player can **name the secret** and *steal it at the buzzer* (+3); mis-accuse and they walk. **Match play**: set **Play to** a target score and the first there takes the crown on a **champion screen** with **MVP** and **confetti**, plus a **share-result** card. Persistent **per-player stats** (win rate, imposter rate, catches, steals) tracked on-device, a **custom pack editor** to build your own categories, and optional **✦ Live AI** categories that have **Claude** generate a fresh pack from any theme via a zero-dependency local proxy ([`imposter/server.mjs`](./imposter/server.mjs), key stays server-side). 15 word packs plus **Mixed**, **1–N imposters** (auto-capped so the crew always outnumbers them), **decoy mode**, sound, haptics. The deal / vote-tally / scoring / standings / stats core is a pure, deterministic, unit-tested engine ([`imposter/engine.js`](./imposter/engine.js), `npm run test:imposter` — 20 tests). Fully offline, **installable PWA**.
- **⭐ [Cortex — the daily brain gym](https://refayethossain28.github.io/BallrzAPP/cortex/)** — the addictive mental exercise game every human needs: **five adaptive drills, one per cognitive domain** — 🧩 **Flashback** (a grid flashes, tap back the lit cells from memory), ➗ **Number Storm** (rapid mental arithmetic against a shrinking clock), 🎯 **Ink Trap** (the classic **Stroop** interference task — answer the *ink*, not the word, with reverse-Stroop mode switches at higher levels), 🔁 **Echo** (a vanishing symbol stream held in working memory, with confusable-twin decoys ◆/◇ once you're good), and ⚡ **Odd One Out** (visual search on an ever-bigger wall of ever-more-similar glyphs). Every session is 10 rounds on a **psychophysical staircase** — hit → level up, miss → level down — so every player converges on the "hard but doable" edge where training actually happens. Points reward speed, a hit-streak **combo multiplier** (up to ×2) rewards consistency, and each session feeds a per-domain **rating** (an EMA that gains fast and decays gently) which averages into your **Brain Index** and named rank (*Spark → Limitless*). A 🔥 **daily streak** rewards showing up, and `dailyWorkout()` deals the **same 3-drill circuit to every human on Earth for a given date** — today's workout is everyone's workout. Every round dealt, answer checked, point scored and rating moved comes from the pure, seeded, deterministic **Drill Engine** ([`cortex/engine.js`](./cortex/engine.js), `npm run test:cortex` — 33 tests). And it's the suite's second **subscription product**: the daily workout, streak and Brain Index are free forever, while **✦ Cortex Pro** (£3.99/mo, 7-day free trial) unlocks unlimited free play — the membership lifecycle (trial → renewal invoices → cancel-at-period-end → resume) and the free-play quota are pure, unit-tested engine logic, the demo billing runs entirely on-device, and the **real Stripe Billing backend is built in** (hosted Checkout, Billing Portal, and a webhook — [`functions/src/cortex.ts`](./functions/src/cortex.ts) — that mirrors subscription truth into Firestore so the client can never grant itself Pro; [`cortex/SETUP.md`](./cortex/SETUP.md) has the 3-step setup). No account needed, fully offline, **installable PWA**.
- **⭐ [ApexVIP Concierge — the all-in-one VIP concierge](https://refayethossain28.github.io/BallrzAPP/concierge/)** — a **subscription-based** personal concierge: choose **Silver (£49/mo), Gold (£199/mo) or Black (£499/mo)** with a **7-day free trial**, then ask for anything across **8 desks** — ✈️ travel, 🍽️ dining, 🎟️ events & tickets, 🚗 chauffeur, 🛍️ personal shopping, 🏠 home & errands, 💆 wellness, 🎁 gifting. Each request runs a real concierge lifecycle in a live chat thread — *received → with your concierge → sourcing → **three priced options** → confirmed → completed* — with **tier-based first-response SLAs** (Black: 15 minutes, and it jumps every queue), monthly request quotas (Black is unlimited), **prorated instant upgrades**, downgrades scheduled at renewal, cancel-at-period-end with resume, full **billing history**, and **ApexVIP points & status** (Member → Insider → Icon → Legend, 1 pt per £1 × tier multiplier). Every money/entitlement rule — trials, renewal invoices, proration, quota, the request state machine, SLA states, priority ordering, even the deterministic desk simulator — lives in the pure, unit-tested **Membership Engine** ([`concierge/engine.js`](./concierge/engine.js), `npm run test:concierge` — 20 tests). **The full cloud backend is built in**: add nothing and it's a pure offline demo; with the Firebase project in [`concierge/config.js`](./concierge/config.js) it gains **real accounts** (email/password or guest), live **cross-device sync** of your membership and requests over Firestore (owner-scoped rules in [`firestore.rules`](./firestore.rules); an admin can staff the desk and reply in real time), and **real subscription payments via Stripe Billing** — hosted Checkout with the same 7-day trial, the Billing Portal for plan changes, and a webhook ([`functions/src/velvet.ts`](./functions/src/velvet.ts)) that mirrors subscription truth into Firestore so the client can never grant itself a tier ([`concierge/SETUP.md`](./concierge/SETUP.md) has the 3-step setup). - **⭐ [Ripple — private messenger](https://refayethossain28.github.io/BallrzAPP/ripple/)** — a messaging app like WhatsApp, but one that finally puts you in control. **Schedule** messages to send later, **edit** and **unsend** them, set a chat to **auto-disappear**, run inline **polls**, fire **slash commands** (`/poll`, `/remind 10m …`, `/expire 1h`, `/me`, `/shrug`, `/shout`, `/clear`), write rich text (`*bold*`, `_italic_`, `` `code` ``), record voice notes, send photos, and **search every chat instantly**. Reactions, threaded replies, delivery & read ticks, typing indicators, pin / mute / archive, starred messages, per-chat wallpapers and accent. Optional **App Lock** encrypts everything on the device (AES-GCM, PBKDF2). It's alive out of the box: message the built-in **Echo** auto-responder, or open two tabs to chat live over `BroadcastChannel` — **no account needed to try it**. The product logic (search ranking, disappearing/scheduled dispatch, poll tally, reaction toggling, sync de-duplication) lives in [`ripple/engine.js`](./ripple/engine.js) and is deterministic and unit-tested (`npm run test:ripple`). **Real cross-device messaging is built in**: drop a Firebase config into [`ripple/config.js`](./ripple/config.js) and Ripple signs people in anonymously and syncs chats live over Firestore — create a cloud chat, **share an invite link**, and anyone who opens it joins and can message you (member-scoped security rules included; [`ripple/SETUP.md`](./ripple/SETUP.md) has the 3-step setup and production hardening). Offline-first, **installable PWA**.
- **🎴 [Intro — your digital business card](https://refayethossain28.github.io/BallrzAPP/intro/)** — retire the paper card. Design a digital business card on your phone — name, role, company, tagline, photo, phone/email/website/location, up to 8 **social profiles** (paste a handle, an @handle or a full profile URL and it's canonicalised), and **8 card themes** with a live preview and a **completeness meter**. Then hand it to anyone as a **link or a QR code**: the whole card is packed into a compact versioned token that travels *inside the link's #fragment* — **no server, no database, no account**, for you or for them. Any phone camera scans the QR and opens your card — and Intro has a **built-in 📷 scanner** (live camera where the browser supports it, paste-the-link everywhere) to pull someone else's card straight into your wallet, plus **📳 NFC tap**: from Chrome on Android, write your card onto any cheap NFC tag or sticker — after that, *any* phone tapped on it (iPhones included) opens your card automatically, exactly how the commercial tap-card products work — with tag-size guidance and a Compact mode for small tags, and tag *reading* built into the scanner; the receiver taps **Save to contacts** and gets a real **vCard 3.0** (RFC-escaped, line-folded, photo embedded) straight into iOS/Android contacts, or keeps you in their in-app **wallet** — and can fire your card right back. There's even **🎟 Apple Wallet**: a complete zero-dependency **.pkpass pipeline** — pass.json, store-ZIP, SHA-1 manifest and a hand-assembled **PKCS#7/CMS signature** ([`intro/pass.mjs`](./intro/pass.mjs), `npm run test:intropass` — 17 tests, verified against a self-built certificate) served by [`intro/server.mjs`](./intro/server.mjs); bring your Apple Developer Pass Type ID cert ([`intro/WALLET.md`](./intro/WALLET.md)) and the button serves a real signed pass whose QR opens your live card. Keep **multiple cards** (work / side-hustle / personal), photos are resized on-device, and hostile link input is neutralised on decode. Every serious bit — the card model, the share codec (from-scratch UTF-8 + base64url), the vCard generator, the **from-scratch QR encoder** — lives in the pure, deterministic **card engine** ([`intro/engine.js`](./intro/engine.js), `npm run test:intro` — 34 tests). Offline, **installable PWA**.
- **⭐ [Drip — the passive income engine](https://refayethossain28.github.io/BallrzAPP/drip/)** — you can't code money into existence, but you *can* build the machine every real passive income stream secretly is: **capital → income → back into capital**. Drip models every stream — 🏦 savings, 📈 dividends, 🧾 bonds, 🏠 rental, 🎼 royalties, 💾 digital products, or anything custom — with one honest, unified parameterisation (capital, yield, income growth *or decay*, appreciation, monthly contributions, DRIP reinvestment, and **honest upkeep hours**, because "passive" is a spectrum). The deterministic **passive income engine** ([`drip/engine.js`](./drip/engine.js)) runs the month-by-month compounding simulation and answers the questions that matter: what does the drip pay **this month**, what will it pay in 10–30 years (nominal *and* inflation-adjusted), what % of your bills it already covers, and — the headline — your **🏁 freedom date**: the first month projected passive income covers your inflation-growing expenses, drawn right on the projection chart. Plus a **passivity score** (income-weighted, with your effective £/hr on upkeep) and a **diversification score** (1 − Herfindahl concentration), because concentration is the classic passive-income failure mode. A planner, not a promise — every figure is your own editable assumption, and the arithmetic is pinned by unit tests (`npm run test:drip` — 23 tests). No account, offline, **installable PWA**.
- **⭐ [Cusp — what to do right now](https://refayethossain28.github.io/BallrzAPP/cusp/)** — the decision engine your to-do list is missing. Every other app *lists* your tasks; Cusp *decides*: of everything on your plate, what should you do **right now**? The on-device **Salience Engine** (a proprietary, fully-specified scoring algorithm — see [`cusp/engine.js`](./cusp/engine.js)) scores every task across six fields — importance, deadline *tightness* (effort ÷ time left, not just time left), how its mental load fits your current energy, whether it fits the minutes you actually have free, momentum (context-switch cost), and staleness — then **shows its working** so the recommendation is auditable, not a black box. Add tasks, set "I've got 25 minutes" and your energy, and Cusp surfaces the one thing, a right-now plan that packs your window, a focus timer, dependency gating, and a decision-debt nudge for tasks you keep skipping. Deterministic and unit-tested (`npm run test:cusp`), offline, no account, **installable PWA**.
- **⭐ [Omni — do-everything app](https://refayethossain28.github.io/BallrzAPP/omni/)** — one offline-first home for your day: tasks & calendar, notes, calculator, unit/currency converter, timers & pomodoro, habit streaks, money log, weather, text tools, password generator, **QR codes** (a from-scratch encoder, verified bit-for-bit against a reference library) and a world clock — with a ⌘K command palette to jump anywhere. No account, no cloud, **installable PWA**, everything stored on-device. Pure logic is covered by `npm test`.
- **🗣️ [Lingua — the fluency engine](https://refayethossain28.github.io/BallrzAPP/lingua/)** — learn **every language in the world**: 180+ built in (listed alphabetically, all treated equally) plus a free-type option for anything else, with regional dialect/variety support for several languages. Built on the five research-backed fluency levers: an **SRS memory deck** (SM-2 spaced repetition), **pronunciation-scored speaking drills** (on-device), **AI-graded production drills**, **graded stories** at your level, and a **streak + daily XP** habit loop — all arranged into a **CEFR course (A1 → B2, 24 units)**. Includes a dedicated **📖 Qur'an studio**: read (verified Tanzil Uthmani text + translation + real per-ayah recitation), memorize with its own hifz SRS deck, recite with on-device scoring, and AI study explanations — scripture is never AI-generated. Runs offline with a starter phrasebook; enable **Live AI** (paste your own Anthropic key, run the local proxy [`lingua/server.mjs`](./lingua/server.mjs), or deploy the hosted function) for full accuracy. **Installable PWA**.
- **⭐ [My Own AI Model — a GPT from scratch](https://refayethossain28.github.io/BallrzAPP/llm/)** — not an API wrapper, a **real language model built from first principles**. The autograd engine, the transformer (causal self-attention), the Adam optimizer and the training loop are all hand-written in pure NumPy — nothing hidden behind a framework ([`llm-from-scratch/`](./llm-from-scratch/)). It trains offline on a small corpus, then `export_web.py` bakes the learned weights into the page and [`web/gpt.js`](./llm-from-scratch/web/gpt.js) re-implements the forward pass in JavaScript (verified to match Python's logits to float32), so **every token is generated on your device — no server, no API, no cloud**. Type a prompt, set temperature / top-k / length, and watch it stream token by token. The autograd is checked against finite-difference numeric gradients (`python llm-from-scratch/test_autograd.py`). **Installable PWA**, fully offline. Trained on the ~1 MB tiny-Shakespeare corpus with a from-scratch **BPE** tokenizer, so it produces real words and Shakespearean cadence — though, being ~1.7M params, it's a stylistic echo, not a smart assistant (a frontier model has ~5–6 orders of magnitude more scale). For real intelligence in the same app, `server.mjs` adds an optional **⚡ Live AI** toggle that streams from **real Fable&nbsp;5** (`ANTHROPIC_API_KEY=sk-ant-… node server.mjs`, key stays server-side) so you can compare your hand-built model against the state of the art side by side.
- **🧠 [Neura — the AI cryptocurrency](https://refayethossain28.github.io/BallrzAPP/neura/)** — **the chain that thinks**: an AI-native cryptocurrency designed as a **store of value**, built on the same from-scratch Bitcoin engine as TimeCoin plus one new consensus rule — **Proof of Intelligence**. Every block must advance the chain's shared **neural network** by one exact, bit-deterministic training step (mini-batches seeded by the previous block's hash, so they can't be cherry-picked) and commit the weights' SHA-256 in its coinbase, sealed under the proof-of-work; **every node re-runs the training to verify it**, so the learning can't be faked or skipped, and a heavier chain with fake intelligence is rejected outright. The network is learning to paint its own mark — open the app and watch the **mind of the chain** (rendered live from the consensus weights) sharpen with every block you mine, tracked by a **synapse score** and a loss chart. Money side: **21,000,000 NEURA hard cap**, Bitcoin's exact 50-per-block/210,000-block halving curve, **no premine** (genesis pays nobody), fees to miners, difficulty retargeting, multi-tab node gossip. The brain ([`neura/brain.js`](./neura/brain.js)) uses only IEEE-exact float ops (ReLU, MSE, integer PRNG — no `Math.exp`/`sin`, which differ across JS engines and would fork the chain), so all nodes derive bit-identical minds; the whole PoI layer is attacked directly in tests — missing, forged and stale commitments all die (`npm run test:neura` — 17 tests). The design, the store-of-value thesis and the honest limitations are in [`neura/README.md`](./neura/README.md) — a working prototype, not an investment.
- **🪙 [TimeCoin — my own cryptocurrency](https://refayethossain28.github.io/BallrzAPP/coin/)** — a complete **Bitcoin-style proof-of-work cryptocurrency built from raw bytes up, with zero dependencies**: SHA-256 implemented from FIPS 180-4, ECDSA on **secp256k1** (Bitcoin's exact curve, with RFC 6979 deterministic signatures and compressed public keys), base58check addresses, a **UTXO ledger**, merkle trees, **proof-of-work mining with Bitcoin-style difficulty retargeting**, a block reward **halving every 210,000 blocks** toward a hard cap — **21 billion TIME, fixed forever**, sized for a worldwide community that trades it for time and favours — transaction fees paid to miners, a fee-priority mempool, and **fork choice by cumulative work** (real Nakamoto consensus). Create wallets, **⛏ mine blocks live in your browser**, send coins and watch them confirm in the block explorer — then open a second tab: **each tab is an independent node**, gossiping blocks and transactions over `BroadcastChannel` and converging on the heaviest chain. The whole consensus core is a pure engine ([`coin/engine.js`](./coin/engine.js)) verified against published crypto test vectors (FIPS 180-4, RFC 4231, the RFC 6979 secp256k1 vector) and attacked directly in tests — double-spends, forged signatures, tampered blocks, coinbase inflation and doctored forks are all rejected (`npm run test:coin` — 39 tests). What's real, and its honest security posture, is documented in [`coin/README.md`](./coin/README.md) — a real community currency for time and favours, not an investment.

## Running locally

The web prototypes are zero-build single-file HTML — clone and open:

```sh
git clone https://github.com/refayethossain28/BallrzAPP.git && cd BallrzAPP
open imposter/index.html                    # pass-and-play imposter party game (3–12 players)
open ripple/index.html                     # WhatsApp-but-better private messenger
open cusp/index.html                       # "what to do right now" — the Salience Engine
open concierge/index.html                  # ApexVIP Concierge — subscription-based
open omni/index.html                       # the do-everything super-app
open lingua/index.html                     # learn & translate any language (+ dialects)
open concepts/prototypes/flow-game/index.html
```

Live AI translations & lessons in Lingua (key stays server-side):

```sh
cd lingua
ANTHROPIC_API_KEY=sk-ant-... node server.mjs   # then open http://localhost:8788
```

Optional **Live AI categories** in Imposter (the game is fully playable without it — this only powers the “generate a pack from a theme” button; key stays server-side):

```sh
ANTHROPIC_API_KEY=sk-ant-... node imposter/server.mjs   # then open http://localhost:8790
```

Live Claude tool-use in the concierge prototype:

```sh
cd concepts/prototypes/concierge-split
ANTHROPIC_API_KEY=sk-ant-... node server.mjs   # then open index.html, toggle "Live"
```

The Concept 1 native iOS slice (SwiftUI + a `swift test`-able kernel) opens in
Xcode 15+ — see [its README](./concepts/concept-1-concierge/).
