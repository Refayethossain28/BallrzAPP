Proof of Intelligence

Every block must think before it counts.

Bitcoin proved that energy can secure money. Neura asks that energy to do one more thing on its way through: teach the chain. A block that didn't train the mind — or trained it wrong by a single bit — is rejected by every node on the network.

The seed is dealt

The training batch for block H is derived from sha256(H | prevHash) — fixed by the chain itself before mining starts. No cherry-picking, no shortcuts.

The miner thinks

One exact SGD step advances the shared network. Bit-deterministic by construction: only IEEE-exact math, so every CPU on Earth derives the same weights.

The work seals it

The new mind's hash is committed in the coinbase — PoI1|… — which feeds the merkle root, which feeds the block hash the proof-of-work grinds against.

Everyone verifies

Every node re-runs the training step and compares hashes. A heavier chain with fake intelligence is rejected outright — work cannot buy a lie.

Absolute scarcity

Money holds value when no one can print it.

  • 21,000,000 NEURA, ever. The cap is a consensus rule every node enforces — not a promise, not a policy, not a committee's mood.
  • Bitcoin's exact emission. 50 per block, halving every 210,000 blocks. The curve on the right is computed live from the same parameters the chain enforces.
  • No premine, no founders' cut. The genesis block pays nobody. Every coin that will ever exist enters the world the same way: someone mined it — and trained the mind to do so.
  • Integer money. Balances are whole sparks (0.00001 NEURA) — no floating-point, no rounding, no leaks.
Circulating supply by block height — each step fainter as the reward halves, forever approaching, never reaching, 21,000,000.
