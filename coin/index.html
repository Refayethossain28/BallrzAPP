BallrzCoin BLZ

A real proof-of-work cryptocurrency built like Bitcoin: secp256k1 wallets, UTXO transactions, merkle trees, difficulty retargeting and halving rewards — a community currency you earn by doing favours and spend on time and favours. 21 billion BLZ will ever exist, hard-capped and fixed forever. This is your wallet — it holds and spends but can’t mint; issuing coins is a separate ⛏ miner’s job. Open this page in a second tab — each tab is a node, and they sync on the heaviest chain.

🔒 Key security
Encrypts your private keys at rest (AES-GCM) so they aren’t readable from browser storage. You’ll unlock with your passphrase each time you open the app. There is no recovery if you forget it — keep a paper wallet, recovery phrase or backup file as your safety net.
💾 Backup
Downloads one encrypted .blzwallet file holding every wallet key on this device — email it to yourself or keep it on a USB stick. If this browser is ever cleared or the device is lost, Restore from file brings every coin back. Open it any time with your backup passphrase.
🔎 Code fingerprint checking… How to verify ›
A fingerprint of the exact code your browser loaded. Everyone running the same version sees the same value — compare yours with a friend, or with the figure announced for the current release on the project page. If it changes when you didn’t update, or differs from everyone else’s, the site may have been tampered with — don’t type your keys. Read the safety guide ›

Barter — offers board

BLZ is a time bank: it buys favours and time, never cash. Post something you’ll do or give, and pay others for theirs. Offers travel the network like payments. 📖 How to run barter with friends →
⏳ The group’s time rate 1 BLZ = min ()
Everyone in your circle should set the same rate — that shared agreement is what gives BLZ its worth. Anchoring to time (not pounds) keeps it a favours co-op, not an investment. Pick a rate that fits your group: fewer minutes-per-BLZ stretches the 21-coin supply across more hours of favours.
💡 Suggested rates — tap any to start an offer
Prices are the hours of effort a favour takes. A starting point — screenshot it and let your group agree its own rates.

Deals

A payment plus a promise. Pay pays the seller directly, then you confirm you got it. 🛡 Escrow pays a trusted third party who releases to the seller once you confirm — protection if you don’t fully trust them yet. Confirming delivery builds the seller’s ✓ reputation. Tap 💬 on any deal to chat with the other party — arrange the details, and keep a signed record if there’s ever a dispute.

Mutual credit beta

The other way to run a community currency: no mining, no fixed supply. Everyone starts at zero; paying someone moves you negative and them positive, so all balances always sum to zero. “Money” appears the moment a favour is done. A shared credit limit caps how far below zero anyone can go — an interest-free overdraft you extend each other.
🤝 Group credit limit hours of credit
Everyone should agree the same limit. It’s how much a member can owe the group before they must earn some back — trust, made explicit.

Network

Tabs in this browser sync automatically. To sync with other devices, run a relay (see the 5-minute deploy guide) and paste its URL here — or open the relay's own URL, which serves this app pre-connected. Add more than one for resilience: the app broadcasts to and polls every relay, so if one sleeps or fails the network keeps running.
